# Recidiviz — адаптивный лендинг

**Демо:** [https://andreygentokarev25-beep.github.io/recidiviz/](https://andreygentokarev25-beep.github.io/recidiviz/)

Вёрстка многосекционного лендинга по макету Figma. Проект выполнен в рамках портфолио фронтенд-разработчика.

## Особенности
- Полное соответствие макету (pixel-perfect для десктопа 1280px)
- Семантическая вёрстка с использованием БЭМ
- Адаптивность под планшеты и мобильные устройства
- Чистый CSS с переменными, Flexbox и Grid
- Абсолютное позиционирование элементов в сложных секциях
- Базовая кроссбраузерность (Chrome, Firefox, Safari, Edge)

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, CSS-переменные)
- JavaScript (минимальный, планируется расширение)
- Методология БЭМ
- Google Fonts (Libre Baskerville + Public Sans)
- GitHub Pages для публикации
